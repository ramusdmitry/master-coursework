# План проекта
DL-сервис краткосрочного прогнозирования криптовалют

## 1. Цель  
Разработать стратегию торговли для  для краткосрочного (прогнозирования цен на криптовалюты.

## 2. Выбор активов
Bitcoin (BTC) и Ethereum (ETH) — как наиболее ликвидные и стабильные активы, три дополнительные криптовалюты из числа топ-5 по объёму торгов.

## 3. Основные задачи  
1. Собрать и верифицировать исторические данные по выбранным активам, собрать фичи (в том числе EMA, MA, RSI).  
2. Собрать baseline - LSTM (и что-то подобное по сложности)
3. Работа по улучшению результата бейзлайна - RL/self-supervised learning/contrastive learning.  
4. Оценить качество прогнозов, сравнить по Sharpe ratio и т.д.

## 4. Методология  
- Данные: OHLCV + добавленные фичи
- Модели:  
  - Базовая — LSTM;  
  - Основная — RL/self-supervised learning/contrastive learning
- Оценка: смотрим Sharpe ratio

## 6. Ожидаемые результаты  
- Рабочий сервис с открытым API для получения прогнозов по выбранным криптовалютам, доступный для демонстрации, техническая документация.
- Выводы о применимости современных архитектур DL для краткосрочного прогнозирования на крипторынке.
